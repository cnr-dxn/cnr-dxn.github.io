Web-based Online Task Management App

Description

A task management application that allows users to create tasks with aspects such as due date, duration, start time, end time, and location, as well as share tasks with other users on the app. Utilizes React-based front-end framework, PostgreSQL data storage, a NextJS hosted server, and Docker-hosted database storage.

Components

Task Creation

Users can create tasks with components such as due date, duration, start time, end time, and location, all of which are stored in the database in their respective data fields, making it easy and efficient for tasks to be created and shared within the app.

Sharing Capable

Users are able to add friends on the app, and once added, are able to share various tasks with their friends, of which will show up on all parties’ respective calendars and to-do lists.

Modern and Sleek Front-End Design

Due to the use of efficient React coding and the use of imported packages, the app has quite a sleek and simplistic look, yet not one that detracts from the overall utility and usability of the app.
